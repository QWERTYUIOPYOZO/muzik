What's BasicComp?

BasicComp is high-quality VST3 compressor plugin that is focused to compress dynamic range naturally. It is suitable for mastering, vocal track, and drum track.

Installation

Please copy BasicComp.vst3 and BasicComp64.vst3 to the following location.

  • C:\Program Files (x86)\Common Files\VST3\BasicComp.vst3
  • C:\Program Files\Common Files\VST3\BasicComp64.vst3

If your DAW doesn't support VST3, please copy BasicCompVst2.dll and BasicComp64Vst2.dll to the following locations.

  • C:\Program Files (x86)\VstPlugins\BasicCompVst2.dll
  • C:\Program Files (x86)\VstPlugins\BasicComp64Vst2.dll

User Interface

1. Fixed Volume Settings
2. Ratio Settings
3. Speed Settings
4. Stereo Expander Settings
5. Input Volume(dB)
6. Output Volume(dB)
7. Loudness compression curve
8. Threshold
9. Fixed Volume in chart

Notes

  • Please enable delay compensation in DAW because BasicComp has some latency.
